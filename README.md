# notes-cn-wcschedule

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面主要用于保存和展示一些静态的信息内容，不针对任何特定网站或域名，也不构成任何形式的服务或产品。

仓库以笔记和存档为目的，内容以 HTML 格式呈现，可直接在浏览器中打开查看。所有页面均为独立文件，不依赖后端或数据库。

## 目录说明

```
notes-cn-wcschedule/
├── index.html          # 首页或导航页（可选）
├── page-1.html         # 示例页面1
├── page-2.html         # 示例页面2
└── assets/             # 静态资源（样式、图片等）
    └── style.css
```

- 根目录下的 `.html` 文件即为独立页面，可直接访问。
- `assets/` 目录存放页面所需的样式表、图片或其他辅助资源。
- 目录结构可能随时间调整，以实际内容为准。
